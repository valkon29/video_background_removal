### Выбор модели
Я остановил свой выбор на MediaPipe Selfie Segmentation.
Это легковесная модель от Google, которая позволяет в реальном времени отделять человека от фона. Основана на MobileNetV3-like с собственными декодерными блоками.

Более подробно, это CNN вида энкодер-декодер. В энкодере использует Depthwise Separable Convolutions, в декодере Transpose Convolutions. Присутствуют Skip Connections, которые объединяют карты признаков из энкодера с выходами декодера.

Размер модели составляет всего около 0.5 Мб. По скорости она также должна удовлетворять нашим требованием, в файле по ссылке приведены некоторые замеры: [qualcomm/MediaPipe-Selfie-Segmentation](https://huggingface.co/qualcomm/MediaPipe-Selfie-Segmentation/blob/main/README.md?code=true)

Кроме того, она оптимизирована для работы на CPU и имеет простую интеграцию с OpenCV

Ссылка на официальный сайт: https://chuoling.github.io/mediapipe/solutions/selfie_segmentation.html

### Реализация приложения
Создано приложение с следующими функциями:

**Основные возможности:**
- Захват видео с веб-камеры в реальном времени
- Сегментация человека/переднего плана
- Три режима замены фона:
  1. Зеленый фон (однотонный цвет)
  2. Размытие фона
  3. Замена на изображение
- Отображение маски сегментации
- Измерение и отображение FPS

**Технические особенности:**
- Работа на CPU (без требований к GPU)
- Поддержка различных разрешений
- Горячие клавиши для управления
- Детальная настройка через аргументы командной строки

## Инструкция по запуску

1. Установка зависимостей:
```bash
pip install -r requirements.txt
```

2. Непосредственно запуск (обрабатывает поток с камеры в real time):
```bash
python src/background_removal.py
```

Параметры запуска:
- `--camera`: ID камеры (по умолчанию: 0)
- `--width`: Ширина кадра (по умолчанию: 640)
- `--height`: Высота кадра (по умолчанию: 480)
- `--mode`: Режим фона: `green`, `blur`, `image`
- `--bg_image`: Путь к фоновому изображению (для режима `image`)
- `--show_fps`: Показывать FPS на экране

Горячие клавиши:
- `q` - выход
- `g` - зеленый фон
- `b` - размытый фон
- `f` - показать/скрыть FPS
